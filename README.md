# Agent PentaFlow
**Multi-Agent Research Report Automation System**

Agent PentaFlow is a modular multi-agent system built with the Google Agent Development Kit (ADK) patterns in mind. It automates the creation of research reports from topic exploration through drafting, visualization, editing, and export.

## Features
- Topic exploration and summary
- Outline generation
- Structured report drafting
- Chart specification generation
- Final editing and export to Markdown/PDF
- Validation checkers for quality assurance

## Quickstart
1. Clone the repo
2. Install dependencies (if any)
3. Run `python scripts/run_pipeline.py --topic "Your topic here"`

## Repository structure
See top-level directories for `src/`, `docs/`, `examples/`, and `scripts/`.
